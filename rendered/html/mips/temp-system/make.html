6.22. Make-4.2.1

The Make package contains a program for compiling packages.

6.22.1. Installation of Make

Prepare Make for compilation: 

./configure \
    --prefix=/tools \
    --build=${CLFS_HOST} \
    --host=${CLFS_TARGET}

Compile the package: 

make

Install the package: 

make install

Details on this package are located in Section 10.55.2, “Contents of Make.”
