6.18. Gawk-4.1.4

The Gawk package contains programs for manipulating text files.

6.18.1. Installation of Gawk

Prepare Gawk for compilation: 

./configure \
    --prefix=/tools \
    --build=${CLFS_HOST} \
    --host=${CLFS_TARGET}

Compile the package: 

make

Install the package: 

make install

Details on this package are located in Section 10.43.2, “Contents of Gawk.”
