13.3. Making the CLFS System Bootable

Create a silo.conf file defining Silo's boot menu: 

cat > /etc/silo.conf << "EOF"
# Begin /etc/silo.conf
partition = 1         # Boot partition (= root partition)
root = /dev/sda1      # Root partition
timeout = 150         # Wait 15 seconds before booting the default section

image = /boot/clfskernel-4.9.21
  label = linux
# End /etc/silo.conf
EOF

Now to make Silo bootable: 

silo -f
