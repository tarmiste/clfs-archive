1.3. Recommended Build Information

Sparc v8 and earlier

Sparc v8 and earlier processors are not supported by NPTL. Sparc v9 are supported by the Sparc 64 build instructions which utilize NPTL. If you have an UltraSparc processor, you have a Sparc v9 which is 64 Bit.
