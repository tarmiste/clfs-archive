10.69. ELFtoaout-2.3

The ELFtoaout package contains the Sparc ELF to a.out converter.

10.69.1. Installation of ELFtoaout

The following patch fixes some 64 bit issues with elftoaout: 

patch -Np1 -i ../elftoaout-2.3-64bit_fixes-1.patch

Compile the ELFtoaout package: 

make

Install the package: 

make install

10.69.2. Contents of ELFtoaout

Installed programs: elf2aout

Short Descriptions

elftoaout

Converts an ELF-formatted file to Sparc a.out format for use with openprom
