10.82. Expat-2.2.0 32 Bit Libraries

Expat is a stream-oriented XML parser library written in C.

10.82.1. Installation of Expat

Prepare Expat for compilation: 

CC="gcc ${BUILD32}" CXX="g++ ${BUILD32}" \
./configure \
    --prefix=/usr \
    --libdir=/usr/lib

Compile the package: 

make

To test the results, issue: 

make check

Install the package: 

make install

Details on this package are located in Section 10.86.2, “Contents of Expat.”
