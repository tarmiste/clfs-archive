10.58. XML::Parser-2.44

XML::Parser is a perl module for parsing XML documents.

10.58.1. Installation of XML::Parser

Prepare XML::Parser for compilation: 

perl Makefile.PL

Compile the package: 

make

To test the results, issue: 

make test

Install the package: 

make install
