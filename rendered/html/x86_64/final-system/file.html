10.63. File-5.31 32 Bit Libraries

The File package contains a utility for determining the type of a given file or files.

10.63.1. Installation of File

Prepare File for compilation: 

CC="gcc ${BUILD32}" \
./configure \
    --prefix=/usr

Compile the package: 

make

To test the results, issue: 

make check

Install the package: 

make install

Details on this package are located in Section 10.64.2, “Contents of File.”
