10.13. MPFR-3.1.5 32 Bit Libraries

The MPFR library is a C library for multiple-precision floating-point computations with correct rounding.

10.13.1. Installation of MPFR

Apply a patch with upstream fixes: 

patch -Np1 -i ../mpfr-3.1.5-fixes-1.patch

Prepare MPFR for compilation: 

CC="gcc -isystem /usr/include ${BUILD32}" \
    LDFLAGS="-Wl,-rpath-link,/usr/lib:/lib ${BUILD32}" \
./configure \
    --prefix=/usr \
    --host=${CLFS_TARGET32} \
    --docdir=/usr/share/doc/mpfr-3.1.5

Compile the package: 

make
make html

Test the results: 

make check

Install the package: 

make install
make install-html

Details on this package are located in Section 10.14.2, “Contents of MPFR.”
