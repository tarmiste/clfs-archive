10.62. Diffutils-3.6

The Diffutils package contains programs that show the differences between files or directories.

10.62.1. Installation of Diffutils

Fix a bug that prevents locale files from being installed: 

sed -i 's:= @mkdir_p@:= /bin/mkdir -p:' po/Makefile.in.in

Prepare Diffutils for compilation: 

CC="gcc ${BUILD64}" \
./configure \
    --prefix=/usr

Diffutils wants ed as the default editor for sdiff. The following sed will change the default to vi: 

sed -i 's@\(^#define DEFAULT_EDITOR_PROGRAM \).*@\1"vi"@' lib/config.h

Compile the package: 

make

To test the results, issue: 

make check

Install the package: 

make install

10.62.2. Contents of Diffutils

Installed programs: cmp, diff, diff3, sdiff

Short Descriptions

cmp

Compares two files and reports whether or in which bytes they differ

diff

Compares two files or directories and reports which lines in the files differ

diff3

Compares three files line by line

sdiff

Merges two files and interactively outputs the results
