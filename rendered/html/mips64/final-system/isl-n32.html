10.22. ISL-0.17.1 N32 Libraries

ISL is a library for manipulating sets and relations of integer points bounded by linear constraints.

10.22.1. Installation of ISL

Prepare ISL for compilation: 

CC="gcc -isystem /usr/include ${BUILDN32}" \
LDFLAGS="-Wl,-rpath-link,/usr/lib32:/lib32 ${BUILDN32}" \
./configure \
    --prefix=/usr \
    --libdir=/usr/lib32

Compile the package: 

make

Test the results: 

make check

Install the package: 

make install

Finally, move a misplaced file: 

mkdir -pv /usr/share/gdb/auto-load/usr/lib32
mv -v /usr/lib32/*gdb.py /usr/share/gdb/auto-load/usr/lib32

Details on this package are located in Section 10.23.2, “Contents of ISL.”
